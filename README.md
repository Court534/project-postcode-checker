# Postcode Checker

A single page website using multiple API's to takes a UK postcode and returns such informatiom as:
- Crimes in the area
- Primary Care Trust
- Nearest Postcodes
- Planning Permission Records


## Tech Stack

**Code:** HTML, CSS, AWS Lambda (Python), jquery

**Server:** AWS S3
